## ✅ 기능 요약
- **데이터 입력**
  - 지시문 작성
  - 입출력 편집  
  
- **데이터 조회**
  - 지시문 전체 조회
  - 입출력 전체 조회
  
- **데이터 검증**
  - 검증하기  
    > 입력에 따른 출력을 유추하며 지시문의 의미가 명확한지 검토하는 기능입니다.
  - 지시문 중복 검사
    > 지시문의 다양성을 보장하기 위한 작업으로, 태스크 내 중복된 지시문이 있는지 검토합니다.
